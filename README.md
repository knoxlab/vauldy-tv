# Vauldy TV

TV client for [Vauldy](https://github.com/knoxmedia/Vauldy) media server. Cross-platform app for **Android TV** and **Apple TV (tvOS)**, built with **react-native-tvos**.

This repository is linked into the main Vauldy project as a git submodule at `Vauldy-TV/`.

## Status

Requirements specification only — implementation not started.

See [doc/电视端需求规格书.md](doc/电视端需求规格书.md).
